# Dasanba Awakening Universe (Static)

部署流程：
1) 將整個資料夾上傳到 GitHub repo（例如 `dasanba-awakening-universe`）。
2) 到 Vercel → Import Git Repository → 選此 repo，Framework 選 Other。
3) 在 Vercel → Settings → Environment Variables 新增：`OPENAI_API_KEY`（你的金鑰）。
4) Deploy；網址會是 `*.vercel.app`。

素材檔案請放入 `assets/` 並維持以下檔名：
- hero.mp4
- ajin.webp / migou.webp / gungun.webp
- trio.webp / trio2.webp / trio3.webp
